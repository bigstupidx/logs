UBUNTU下安装apahce/mysql/php相关

版本：12.10

安装apache2

sudo apt-get install apache2

安装PHP5

sudo apt-get install php5

注：现在已经不在需要安装libapache2-mod-php5了

安装MYSQL

sudo apt-get install mysql-server

让apache，php支持mysql

		sudo apt-get install libapache2-mod-auth-mysql

		sudo apt-get install php5-mysql

安装GD库

sudo apt-get install php5-gd

开启rewrite

		sudo a2enmod rewrite 开启 Mod_rewrite 模块
		vi /etc/apache2/sites-enabled/000-default 找到相应的目录将 AllowOverride none 改为 AllowOverride All;

设置虚拟机

虚拟机的配置文件都放在/etc/apache2/sites-available目录下，里面有个default文件，复制它，修改内容

cd /etc/apache2/sites-available

cp default

vi

default的内容
